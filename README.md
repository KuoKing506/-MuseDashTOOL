# MuseDashModTool (MuseDashTOOL)

[![Maintenance](https://img.shields.io/badge/Maintained%3F-yes-green.svg)](https://github.com/your-username/MuseDashModTool/graphs/commit-activity)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个为《喵斯快跑》(Muse Dash) 设计的现代、功能强大且易于使用的模组管理工具。基于 AvaloniaUI 构建，提供跨平台支持和流畅的用户体验。

## ✨ 特性

-   **模组管理**：一键安装、卸载和更新模组。
-   **配置管理**：直观地修改模组配置文件，无需手动编辑 JSON。
-   **曲谱下载**：集成曲谱库，快速搜索并下载社区制作的曲谱。
-   **MelonLoader 支持**：内置 MelonLoader 安装与环境维护。
-   **现代 UI**：采用 Fluent 设计风格，支持动态布局。

## 🚀 快速开始

### 准备工作

-   确保已安装 `.NET 9.0 SDK`。
-   拥有合法安装的《喵斯快跑》游戏。

### 运行项目

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/MuseDashModTool.git
   ```
2. 进入目录：
   ```bash
   cd MuseDashModTool/MuseDashTOOL
   ```
3. 运行应用：
   ```bash
   dotnet run
   ```

## 🛠️ 技术栈

-   **框架**: [AvaloniaUI](https://avaloniaui.net/)
-   **模式**: MVVM (CommunityToolkit.Mvvm)
-   **依赖注入**: Microsoft.Extensions.DependencyInjection
-   **音频处理**: NAudio
